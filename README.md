Lista de Tareas (Tkinter)
Descripción
Aplicación de escritorio desarrollada en Python con Tkinter para gestionar tareas (añadir, completar y eliminar), utilizando arquitectura modular por capas.

Funcionalidades
Añadir tareas
Marcar como completadas
Eliminar tareas
Evento Enter y doble clic
Cambio visual de tareas completadas

Tecnologías
Python
Tkinter
PyInstaller
